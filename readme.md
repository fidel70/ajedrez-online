# ♟️ Ajedrez Online

Un juego de ajedrez en tiempo real para dos jugadores, implementado con Node.js y Socket.IO.

## 🚀 Características

- Partidas en tiempo real
- Sistema de códigos para unirse a partidas
- Interfaz intuitiva
- Validación de movimientos
- Sin necesidad de registro

## 🛠️ Instalación

1. Clona el repositorio
```bash
git clone https://github.com/tu-usuario/ajedrez-online.git
```

2. Instala las dependencias
```bash
cd ajedrez-online
npm install
```

3. Inicia el servidor
```bash
npm start
```

4. Abre el navegador en `http://localhost:3000`

## 🎮 Cómo jugar

1. El primer jugador crea una nueva partida
2. El sistema genera un código único
3. El primer jugador comparte el código con su oponente
4. El segundo jugador ingresa el código para unirse
5. ¡La partida comienza automáticamente!

## 🔧 Tecnologías utilizadas

- Node.js
- Express
- Socket.IO
- chess.js
- chessboard.js
